ShopEasy is a simple Flutter application developed to demonstrate UI development, state management, and clean project structure using Flutter. The app simulates a basic e-commerce experience where users can browse products, add items to a cart, and view profile information.

This project focuses on clarity, maintainability, and best practices suitable for beginner to intermediate Flutter development.

*Features

Product listing displayed using a responsive grid layout

Add to Cart functionality with real-time cart count updates

Cart state managed using GetX for reactive UI updates

Profile screen displaying user information and past orders

Bottom navigation for smooth screen switching

Clean separation of UI and business logic

*State Management

The app uses GetX for state management.
Cart data and profile information are managed using controllers, ensuring that the UI automatically updates when the underlying state changes.

This approach avoids unnecessary widget rebuilds and keeps the code organized and scalable.
